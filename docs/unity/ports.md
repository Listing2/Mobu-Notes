# 포트·OSC·VMC

아바타별 포트 할당 규칙과 에디터 도구 사용법.

## Network Transform Setup (Unity 에디터)

- **Auto Assign Ports**: 체크된 아바타만 `Base Port + 0, 1, 2...` 순서로 재배정
- 적용 대상: `uOscServer.port`, `TrackingManager.facePort`
- 선택 없으면 버튼 비활성
