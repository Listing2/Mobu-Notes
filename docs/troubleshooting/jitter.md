# 지터 (Jitter) 트러블슈팅

## 증상

- 캐릭터·카메라가 미세하게 흔들림
- 클로즈업·망원·손/머리에서 두드러짐

## 분리 검증

```text
A. 코드/regression (git log, 최근 커밋)
B. 입력 품질 (원본 캡처 fps, 인원 수)
C. 네트워크 (지연, 패킷 간격)
D. VMC/손가락 sender 우회
```

## 체크리스트

- [ ] 원본 캡처 fps (60 vs 120)
- [ ] Mobu 송출 fps
- [ ] 동시 인원 수
- [ ] `vmcFingerActive`로 sender 우회 여부
- [ ] 카메라 트래킹 임시 수정과 무관한지 (neck 등 단일 본 이슈 vs 전역 지터)
